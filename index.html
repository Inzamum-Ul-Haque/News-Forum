    Questions and Answers

    1. Var is hoisted at the top of global scope and can be used before the declaration. Let and var is hoisted at the top of some private scope and is only available after assigning value, also they can't be used before declaration.
    2. Var is global scoped. Let and const are block scoped.
    3. We can redeclare var in the same scope. Let can't be redeclared in the same scope and const can't be redeclared after initialization.

    Regular functions created using function declarations or expressions are constructible and callable. Since regular functions are constructible, they can be called using the new keyword. However, the arrow functions are only callable and not constructible, arrow functions can never be used as constructor functions.

    Foreach takes a callback function and run that callback function on each element of array one by one. Filter executes the callback and returns an array with the condition applied. Map is like filter & foreach which takes a callback and run it against every element on the array but whats makes it unique is it generate a new array based on the existing array. Find is just like filter but it returns only the first element based on the condition applied.

    We use template string because we can put dynamic values inside them and can use multiline string which are hassle in normal quotation string.
